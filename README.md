# gomkdir

## Description
gomkdir is a simple command line tool to create a directory.

## Installation

```bash
$ make build
```

## Available Options

```bash
$ gomkdir -h
```

```bash
Usage of gomkdir:
  -foldername string
        Folder name to create (default "temp")
  -permission string
        Permission to set (default "0755")
```

## Usage

```bash
$ gomkdir -foldername=<FOLDERNAME> -permission=<PERMISSION>
```
